# Kişisel Portföy Sitem

Bu proje, yazılım alanındaki projelerimi, geliştirdiğim uygulamaları ve kendimi tanıttığım kişisel portföy sitesidir.  
ASP.NET Core MVC ile geliştirilmiş, dinamik ve modern bir yapıya sahiptir.  
Projelerim, dosya indirilebilir şekilde yayınlanmakta; iletişim formu ve admin paneli ile site tamamen yönetilebilir durumdadır.

---

## 🚀 Özellikler

- **Projelerim:**  
  Farklı dillerde (HTML, CSS, JS vb.) geliştirdiğim yazılım uygulamalarını dosya halinde ziyaretçilere sunabiliyorum.  
  Her projede detay sayfası, açıklama, ekran görüntüsü ve indirme butonu mevcut.

- **Hakkımda:**  
  Kendi özgeçmişim, teknik bilgilerim ve ilgi alanlarımın bulunduğu bölüm.

- **İletişim:**  
  Ziyaretçiler bana iletişim formu üzerinden mesaj gönderebiliyor.  
  Gelen mesajlar admin paneline kaydoluyor ve oradan görüntülenebiliyor.

- **Admin Paneli:**  
  Proje ekleme, düzenleme, silme; iletişim mesajlarını yönetme ve uygulama dosyalarını yükleme gibi işlemler panelden kolayca yapılabiliyor.

---

## 🛠️ Kullanılan Teknolojiler

- **Backend:** ASP.NET Core MVC, Entity Framework Core
- **Veritabanı:** MSSQL 
- **Frontend:** HTML5, CSS3, JavaScript, Bootstrap 
- **Diğer:**  
  - Dinamik dosya yükleme ve indirme sistemi  
  - Responsive tasarım  
  - Form validasyonları ve veri doğrulama

## 🚀 Kurulum ve Çalıştırma

1. Bu repoyu klonlayın veya ZIP olarak indirin.
2. `appsettings.json` dosyasındaki veritabanı bağlantı ayarlarını kendi ortamınıza göre düzenleyin.
3. Gerekli NuGet paketlerini yükleyin (`dotnet restore` veya Visual Studio’dan).
4. Migration ve veritabanı işlemlerini tamamlayın (`dotnet ef database update`).
5. Uygulamayı çalıştırın:
    ```
    dotnet run
    ```
6. Tarayıcınızda `https://localhost:5001/` veya kullandığınız portu açın.
